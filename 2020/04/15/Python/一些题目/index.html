变量作用域规则

解析：

Python中的变量作用域规则遵循 L-E-G-B 的顺序。其中：

  1. L 是指 Local, 指本地变量, 例如 def 或者 lambda 定义的函数体内的变量
  2. E 是指 Enclosing-function locals, 即闭包变量
  3. G 是指 Global, 全局变量
  4. B 是指 Built-in 即Python解释器提供的内置函数和变量
