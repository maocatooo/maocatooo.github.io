使用 langchain 做一个 prompt 工程师 🙈

langchain是一个开发由语言模型驱动的应用程序的框架,简单来说就是将LMM打包层应用层面的封装,使普通开发者(比如我)也能开发AI相关的应用

我能想到的相关应用：

  1. 聊天机器人
  2. 知识文档问答库(AI客服) Langchain-Chatchat
  3. 构建NLP应用程序
  4. …

如果你使用过 ChatGPT 就会发现 GPT 回答问题的时候就会多说一些前后缀相关无用的东西,使用 langchain 的 prompt template 就能非常方便的避免掉这个问题
